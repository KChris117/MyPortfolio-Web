About the App

Guide Me is a tour guide application designed to provide information about tourist attractions in Batam city. It is equipped with Artificial Intelligence (AI) and Google Maps API for real-time navigation.

The application is built using Dart and Flutter for a seamless cross-platform mobile experience. We utilize JavaScript for the Midtrans API integration, and rely on Firebase and Supabase for a robust database foundation.

Tech Stack

🖥️ Front-End: Flutter & Dart with Provider for cross-platform real-time state management.

🗄️ Back-End & Database: Firebase Auth & Cloud Firestore (NoSQL real-time data). Supabase Storage is utilized for handling large media files. Firebase Cloud Functions (Node.js) securely handles WebHooks.

💳 Payment Gateway: Midtrans API (Snap) via REST API to process QRIS and e-Wallets seamlessly.

🗺️ Maps & Geolocation: Flutter Map (OpenStreetMap) and Geolocator for interactive routing and location coordinates.

Core Features

📍

Interactive Navigation

Determine your position and navigate accurately using Google Maps API. Choose your preferred vehicle and discover the nearest and farthest routes.

🔍

Smart Categorization

Easily find your destination through various categories such as Hotels, Restaurants, Malls, Beaches, Historical Sites, Culture, Religion, and many more.

Review & Community

Share your experience! Users can read the latest situations via blogs and provide valuable reviews for destinations they have visited.

How to Run Locally

Follow these steps to setup the environment on your local machine:

git clone https://github.com/KChris117/GuideMe_Mobile_App.git
cd GuideMe_Mobile_App

// 1. Install local server like XAMPP / WAMP
// 2. Start Apache and MySQL services
// 3. Move the project folder to htdocs / www directory
// 4. Open phpMyAdmin and create a new database "guideme_db"
// 5. Import the provided .sql file into the database
// 6. Adjust the database configuration in your PHP connection file
// 7. Access via browser: http://localhost/GuideME
