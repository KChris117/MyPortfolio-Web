About the App

Our application is built using the Laravel PHP framework, which provides a solid foundation for developing robust web applications with a powerful routing system, database migrations, and authentication.

In addition to Laravel, we've utilized TailwindCSS to rapidly build custom designs. We've also integrated DaisyUI, which extends TailwindCSS with ready-made components like buttons, cards, and navigation bars to create stunning interfaces.

Frequently Asked Questions

What does your website offer?

We make getting tickets to your favorite concerts easy. Browse our curated selection of live music events, choose your seats, and book with confidence.

How to receive my tickets?

After you complete your ticket purchase, you will receive a confirmation email. Your concert tickets will be delivered electronically via email.

Can I cancel my tickets?

Unfortunately, we are unable to offer cancellations or exchanges for concert tickets once they have been purchased.

What payment methods do you accept?

We accept credit/debit cards, PayPal, and other secure payment options to ensure your transaction is safe and convenient.

How to Run Locally

Follow these steps to launch the app on your local machine:

git clone https://github.com/muhammadpadanta/p5_proweb.git
cd p5_proweb

composer install
composer dump-autoload
composer update

// If you encounter an Error Code 500:
1. Rename .env-example to .env
2. Set APP_DEBUG=true in the .env file
3. php artisan key:generate

// Launch The App
php artisan serve
