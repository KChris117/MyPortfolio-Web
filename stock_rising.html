Stock Rising

A digital board game for stock investment simulation built on Android.

💻 View Source Code 🎮 Play Game

The Story Behind Stock Rising

Stock Rising was originally developed as a physical board game designed to educate players on the fundamental concepts of stock investment in a fun and risk-free environment. However, to reach a broader audience and provide a more interactive experience, it has now been transformed into a digital game for Android.

This project focuses on a multiplayer mode that allows real-time interaction among players, bringing the engaging experience of learning about financial literacy to the palm of their hands. Players can trade stocks across four industry sectors, facing dynamic market changes driven by rumors and events, all simulated using a virtual currency.

Developed as a final project (PBL) by students from the D3 Informatics Engineering program at Politeknik Negeri Batam.

Key Features

  • Multiplayer Simulation: Supports 2-5 players in an online environment using Client-Hosted Listen Server architecture.
  • Turn-Based Gameplay: Follow realistic market cycles including Bidding, Action, Sell, Rumor, and Resolution phases.
  • Dynamic Market Events: Influence stock prices using rumor cards or adapt to shifting sector performances and dividends.
  • Avatar & Border Shop: Players can customize their profile by purchasing avatars and borders using "Finpoin" (virtual currency).
  • Social Features: A complete friend system allowing users to search, add, accept, or ignore friend requests.

Tech Stack

Built with Unity Engine using C# and Photon PUN 2 for real-time multiplayer networking.
